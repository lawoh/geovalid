Guide d'utilisation de GéoValid

GéoValid est un outil de validation de données géographiques pour le Canada. Il détecte automatiquement vos colonnes en analysant les valeurs, pas les noms de colonnes. Vous pouvez nommer vos colonnes comme vous voulez !

Les 3 étapes simples

1

Importez votre fichier

Glissez-déposez ou sélectionnez votre fichier. De nombreux formats sont acceptés : CSV, XLS, XLSX, JSON, GeoJSON, KML, GPX...

2

Vérifiez l'aperçu

Consultez les 10 premières lignes et vérifiez que les colonnes latitude/longitude ont été correctement détectées.

3

Analysez & Exportez

Lancez l'analyse, visualisez sur la carte et exportez vos données validées dans le format de votre choix.

Structure des données

Obligatoire

Latitude

Valeur numérique entre -90 et 90. Pour le Canada : 41.67° à 83.11°

Détection auto par les valeurs, peu importe le nom de colonne

Obligatoire

Longitude

Valeur numérique entre -180 et 180. Pour le Canada : -141° à -52.62°

Détection auto par les valeurs, peu importe le nom de colonne

Optionnel

Code postal

Format canadien : A1A ou A1A1A1

Si présent, sera validé et inclus dans les stats

Optionnel

Province

Code 2 lettres : AB, BC, MB, NB, NL, NS, NT, NU, ON, PE, QC, SK, YT

Si présente, sera incluse dans les stats par province

Détection intelligente : L'application analyse les valeurs de chaque colonne pour déterminer automatiquement ce qu'elle contient. Vous n'avez pas besoin d'utiliser des noms de colonnes spécifiques comme "lat" ou "longitude".

Règles de validation

Données valides

  • Latitude entre 41.67° et 83.11°
  • Longitude entre -141° et -52.62°
  • Valeurs numériques correctes
  • Code postal au bon format (si fourni)

Données invalides

  • Coordonnées hors du Canada
  • Valeurs non numériques
  • Latitude ou longitude manquante
  • Format code postal incorrect

Questions fréquentes

Mes colonnes ne sont pas détectées automatiquement

Vérifiez que vos données de latitude sont des nombres entre -90 et 90, et vos longitudes entre -180 et 180. Pour de meilleurs résultats avec des données canadiennes, les latitudes devraient être entre 40 et 85, et les longitudes entre -145 et -50.

Puis-je utiliser des données hors du Canada ?

Actuellement, l'application valide uniquement les coordonnées situées dans les limites géographiques du Canada. Les points hors de ces limites seront marqués comme invalides.

Comment fonctionne l'export Shapefile ?

L'export Shapefile génère un fichier ZIP contenant les 4 fichiers nécessaires : .shp (géométries), .shx (index), .dbf (attributs) et .prj (projection WGS84).

Mes données sont-elles sécurisées ?

Oui ! Toutes les opérations sont effectuées localement dans votre navigateur. Aucune donnée n'est envoyée vers un serveur externe.
